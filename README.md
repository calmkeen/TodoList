# TodoList for ani


## 목표

todoList로써 기본적인 계획이 아닌 사람들의 취미를 리스트화하여 세울수있는 todoList

## 기술군

- swiftUI
- Fireabse
- Realm( 예정 )

사용 스킬

- MVVM
- avkit
- webkit
- foundation
- swiftUI
- uiKit
- combine( 예정 )
- 데이터 가져올수 있게 된다면 json부분도 고려.


## 진행사항

현재는 애니메이션으로 시작하고 있지만 리스트화 할수 있는 취미
- 만화
- 애니
- 소설
- 악기( 노래 )
- 기타등등

을 모두 포함시킬 예정

## 예상되는 문제

데이터를 어디서 가져오게 되는가가 가장 문제인데 현재는 출시와 거리가 먼 앱이기 때문에 상관없지만 
이중 어느 데이터도 쉽게 가져올수있는 곳이없다는 점

## 문제 해결

swiftUI 기반의 개발이다 보니 아직 uiKit에서 기능을 많이 가져다 쓴다. 이에 영상 부분의 유튜브링크를 통해 불러오는 부분이 생각과 다르게 되었다.
avKit -> webKit으로 사용
